##installation

    npm install mdserver -g

```sh

  Usage: mdserver [options]

  Options:

    -h, --help        output usage information
```

## Examples

```sh
# use current dir and default port 3333
mdserver

# set root dir and port
mdserver -p 8788 -r ~/github/mdserver
```
